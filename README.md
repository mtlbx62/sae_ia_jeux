# SAE_jeux - Dernière Nuit au Poste 77 / Last Night at Outpost 77

## Auteurs
[Talbi Marouan] - [Pierrot Alexandra]

Développement en C / Anglais

Jeu : Dernière Nuit au Poste 77 (Horreur survival)


## Description
Jeu d'horreur survival en mode texte. Vous êtes le dernier opérateur d'un poste de surveillance post-apocalyptique. Survivez 8 heures en gérant vos ressources tout en empêchant des entités inconnues d'entrer.

## Fonctionnalités implémentées
- **Partie obligatoire complète** :
  - Sélection de langue (FR/EN)
  - Création de personnage (nom + type)
  - Système de ressources (énergie, santé mentale)
  - Journal des événements
  - Actions avec conséquences
  - Conditions de victoire ou de défaite

# Compilation et exécution (Version : Linux)

## Linux (Terminal)
## Compiler
make

## Exécuter
./poste77

## Nettoyer
make clean

# Version Windows

## Windows:
1. Etre sur que le gcc est executer dans VSC (MinGW recommandé)
2. Double-cliquer compile.bat
3. Executer poste77.exe
